# 上传部署检查清单

## 📋 部署前检查清单

在上传代码之前,请确保完成以下所有检查项:

### ✅ 基础配置检查

- [ ] **AppID配置**
  - 文件: `project.config.json`
  - 确认已替换为你的小程序AppID
  - 不是测试号,是正式的AppID

- [ ] **云开发环境ID**
  - 文件: `app.js`
  - 确认已配置正确的云开发环境ID
  - 格式如: `wedding-7g2h4m5n`

- [ ] **婚礼信息**
  - 文件: `app.js` 中的 `globalData.weddingInfo`
  - ✓ 新郎新娘姓名
  - ✓ 婚礼日期和时间
  - ✓ 酒店名称和地址
  - ✓ 地图经纬度坐标
  - ✓ 联系人和电话

### ✅ 图片资源检查

- [ ] **婚纱照片**
  - 目录: `images/`
  - 至少3-4张照片
  - 每张图片 < 500KB
  - 格式: JPG 或 PNG
  - 文件名与代码中引用一致

- [ ] **TabBar图标** (可选)
  - 目录: `images/tabbar/`
  - 4张图标: home.png, details.png, rsvp.png, blessings.png
  - 尺寸: 81px × 81px
  - 格式: PNG

### ✅ 云开发配置检查

- [ ] **云开发已开通**
  - 在微信开发者工具中点击"云开发"
  - 确认环境已创建成功
  - 环境状态为"正常"

- [ ] **数据库集合已创建**
  - `rsvp` 集合 - 存储回执数据
  - `blessings` 集合 - 存储祝福留言
  - `lottery_numbers` 集合 - 存储抽奖号码
  - `lottery_prizes` 集合 - 存储中奖记录

- [ ] **数据库权限已设置**
  - `rsvp`: 所有用户可读写
  - `blessings`: 所有用户可读写
  - `lottery_numbers`: 仅创建者可读写
  - `lottery_prizes`: 所有用户可读,仅创建者可写

## 🚀 上传步骤

### 第一步: 最后检查

1. **保存所有文件**
   - 确保所有修改都已保存
   - 关闭所有编辑器

2. **清理缓存**
   - 在微信开发者工具中点击"清缓存"
   - 选择"清除全部缓存"
   - 重新编译项目

3. **最后测试**
   - 完整测试一遍所有功能
   - 确认没有问题

### 第二步: 上传代码

1. **点击上传按钮**
   - 在微信开发者工具顶部工具栏
   - 点击"上传"按钮

2. **填写版本信息**
   ```
   版本号: 1.0.0
   项目备注: 婚礼邀请函小程序初始版本
   
   功能说明:
   - 首页: 婚纱照展示、倒计时
   - 详情: 婚礼流程、地图导航
   - 回执: 宾客信息收集
   - 祝福: 留言板功能
   - 抽奖: 现场抽奖功能
   ```

3. **确认上传**
   - 检查版本信息
   - 点击"上传"
   - 等待上传完成

### 第三步: 提交审核

1. **登录微信公众平台**
   - 访问: https://mp.weixin.qq.com/
   - 使用管理员微信扫码登录

2. **进入版本管理**
   - 左侧菜单: 管理 → 版本管理
   - 找到刚上传的版本

3. **填写审核信息**

   **基本信息:**
   - 小程序名称: 婚礼邀请函
   - 小程序简介: 王峰&杨丽琴的婚礼邀请函
   - 服务类目: 生活服务 → 婚庆服务

   **功能页面:**
   - 首页: pages/index/index
   - 详情页: pages/details/details
   - 回执页: pages/rsvp/rsvp
   - 祝福页: pages/blessings/blessings

   **测试账号:** (如需要)
   - 无需测试账号(公开功能)

   **补充说明:**
   ```
   这是一个婚礼邀请函小程序,主要功能包括:
   1. 展示婚礼信息和倒计时
   2. 提供婚礼流程和地图导航
   3. 收集宾客回执信息
   4. 宾客留言祝福功能
   5. 现场抽奖互动功能
   
   所有功能均为婚礼相关服务,不涉及支付、社交等敏感功能。
   ```

4. **提交审核**
   - 确认所有信息无误
   - 点击"提交审核"
   - 等待审核结果

## ⏰ 审核时间线

- **提交审核**: 立即
- **审核中**: 1-7个工作日
- **审核结果**: 通过公众平台消息通知
- **发布上线**: 审核通过后立即发布

## ✅ 审核通过后

1. **发布小程序**
   - 在版本管理中点击"发布"
   - 确认发布

2. **测试线上版本**
   - 在微信中搜索小程序
   - 完整测试所有功能

3. **分享给宾客**
   - 生成小程序码
   - 分享到微信群/朋友圈
   - 或打印在纸质邀请函上

## ❌ 审核不通过怎么办?

### 常见审核问题

1. **缺少隐私政策**
   - 添加隐私政策页面
   - 说明数据收集和使用方式

2. **功能描述不清晰**
   - 完善小程序简介
   - 详细说明各个功能

3. **服务类目不匹配**
   - 修改为正确的服务类目
   - 婚庆服务 或 生活服务

4. **内容违规**
   - 检查是否有敏感词汇
   - 修改后重新提交

### 修改后重新提交

1. 根据审核反馈修改代码
2. 重新上传新版本
3. 再次提交审核
4. 通常第二次审核会更快
