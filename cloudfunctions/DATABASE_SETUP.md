# 数据库集合创建指南

在使用云函数之前，需要先在云开发控制台创建数据库集合。

## 创建步骤

### 1. 打开云开发控制台

1. 在微信开发者工具中点击"云开发"按钮
2. 进入云开发控制台
3. 点击左侧菜单"数据库"

### 2. 创建 rsvp 集合（宾客回执）

1. 点击"添加集合"按钮
2. 集合名称：`rsvp`
3. 点击"确定"创建

**字段说明**：
```javascript
{
  _id: String,              // 自动生成的记录ID
  _openid: String,          // 自动生成的用户openid
  name: String,             // 姓名
  phone: String,            // 电话
  attendees: Number,        // 到场人数（1-4）
  hasChildren: Boolean,     // 是否有儿童
  dietaryRestrictions: String, // 饮食禁忌
  createTime: Date,         // 提交时间（服务端时间）
  updateTime: Date          // 更新时间
}
```

**权限设置**：
- 开发阶段：选择"所有用户可读写"
- 生产环境：建议选择"仅创建者可读写"或使用云函数控制权限

### 3. 创建 blessings 集合（祝福留言）

1. 点击"添加集合"按钮
2. 集合名称：`blessings`
3. 点击"确定"创建

**字段说明**：
```javascript
{
  _id: String,              // 自动生成的记录ID
  _openid: String,          // 自动生成的用户openid
  nickname: String,         // 访客昵称
  content: String,          // 祝福内容
  createTime: Date,         // 发送时间（服务端时间）
  isVisible: Boolean        // 是否可见（预留审核功能）
}
```

**权限设置**：
- 开发阶段：选择"所有用户可读写"
- 生产环境：建议选择"所有用户可读，仅创建者可写"

## 权限设置详解

### 所有用户可读写
- 适用场景：开发测试阶段
- 优点：简单方便，无需额外配置
- 缺点：安全性较低，用户可以修改他人数据

### 仅创建者可读写
- 适用场景：生产环境
- 优点：用户只能读写自己创建的数据
- 缺点：需要配合云函数使用

### 自定义权限
- 适用场景：复杂权限需求
- 使用云函数控制所有数据操作
- 在云函数中实现自定义权限逻辑

## 推荐配置

### 开发阶段
```
rsvp 集合：所有用户可读写
blessings 集合：所有用户可读写
```

### 生产环境
```
rsvp 集合：仅创建者可读写
blessings 集合：所有用户可读，仅创建者可写
```

## 验证集合创建

创建完成后，在数据库页面应该看到两个集合：
- ✅ rsvp
- ✅ blessings

点击集合名称可以查看集合详情和数据记录。

## 常见问题

### Q: 找不到"添加集合"按钮
A: 确保已经开通云开发服务，并且在"数据库"页面

### Q: 创建集合失败
A: 检查集合名称是否符合规范（只能包含字母、数字、下划线）

### Q: 权限设置后无法读写数据
A: 检查权限设置是否正确，开发阶段建议使用"所有用户可读写"

### Q: 数据库容量不足
A: 基础版提供2GB容量，可以升级套餐或清理旧数据
