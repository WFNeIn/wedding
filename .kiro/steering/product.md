# Product Overview

This is a WeChat Mini Program for wedding invitations (婚宴请帖小程序). It provides a digital wedding invitation with guest management features.

## Core Features

- Wedding invitation homepage with photo carousel and countdown timer
- Wedding details page with timeline, map navigation, and contact info
- RSVP system for guest responses
- Blessing/message board for guest wishes
- Lottery system for wedding reception
- Seat selection system with visual layout and admin view

## Target Users

- Couples getting married who want a digital invitation
- Wedding guests who receive the invitation
- Wedding administrators managing RSVPs and seating
