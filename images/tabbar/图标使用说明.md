# TabBar 图标使用说明

## 当前状态

✅ 已添加4张TabBar图标:
- `home.png` - 首页图标
- `details.png` - 详情图标
- `rsvp.png` - 回执图标
- `blessings.png` - 祝福图标

✅ 已在 `app.json` 中配置使用这些图标

## 图标要求

### 微信小程序TabBar图标规范

1. **尺寸要求**
   - 推荐尺寸: 81px × 81px
   - 支持格式: PNG (推荐使用PNG格式,支持透明背景)

2. **颜色要求**
   - 未选中状态: 灰色 (#999999)
   - 选中状态: 天蓝色 (#87CEEB)

3. **文件命名建议**
   - 未选中: `home.png`, `details.png`, `rsvp.png`, `blessings.png`
   - 选中: `home-active.png`, `details-active.png`, `rsvp-active.png`, `blessings-active.png`

## 当前配置

目前配置中,未选中和选中状态使用的是同一张图片:

```json
{
  "iconPath": "images/tabbar/home.png",
  "selectedIconPath": "images/tabbar/home.png"
}
```

这意味着:
- ✅ 图标会正常显示
- ⚠️ 选中和未选中状态看起来一样(除了文字颜色变化)

## 优化建议

### 方案1: 添加选中状态图标(推荐)

如果你想要更好的视觉效果,可以为每个图标准备两个版本:

1. **准备选中状态图标**
   - 复制现有的4张图标
   - 将颜色改为天蓝色 (#87CEEB)
   - 命名为: `home-active.png`, `details-active.png`, `rsvp-active.png`, `blessings-active.png`
   - 放在 `images/tabbar/` 目录下

2. **更新 app.json 配置**
   ```json
   {
     "pagePath": "pages/index/index",
     "text": "首页",
     "iconPath": "images/tabbar/home.png",
     "selectedIconPath": "images/tabbar/home-active.png"
   }
   ```

### 方案2: 保持当前配置(简单)

如果你觉得当前效果已经足够好,可以保持现状:
- 图标颜色不变
- 只有文字颜色会在选中时变为天蓝色
- 这样也是可以接受的设计

## 如何制作选中状态图标

### 使用图片编辑工具

1. **Photoshop / GIMP**
   - 打开原图标
   - 使用"色相/饱和度"调整
   - 或使用"颜色替换"工具
   - 将灰色改为 #87CEEB
   - 导出为PNG格式

2. **在线工具**
   - 使用在线图片编辑器(如 Photopea)
   - 上传图标
   - 调整颜色
   - 下载修改后的图标

3. **AI工具**
   - 使用AI图片编辑工具
   - 提示词: "将这个图标的颜色改为天蓝色 #87CEEB"

### 常见问题

**Q: 图标不显示怎么办?**
A: 检查以下几点:
- 图片路径是否正确
- 图片格式是否为PNG
- 图片尺寸是否合适(推荐81px × 81px)
- 文件名是否正确(区分大小写)

**Q: 图标显示模糊怎么办?**
A: 
- 确保图片尺寸至少为81px × 81px
- 使用2倍图(162px × 162px)效果更好
- 确保图片是PNG格式,不要使用JPG

**Q: 图标颜色不对怎么办?**
A:
- TabBar会自动处理图标颜色
- 建议使用纯色图标(黑色或灰色)
- 微信会根据 `color` 和 `selectedColor` 自动着色
