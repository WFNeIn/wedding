## 如何从 Iconify 下载图标

### 方法一：在线下载 SVG（推荐）

1. 访问 https://iconify.design/
2. 搜索图标名称（如 `mdi:bell-outline`）
3. 点击图标进入详情页
4. 点击右上角"Download"按钮
5. 选择 SVG 格式下载

### 方法二：直接复制 SVG 代码

1. 在图标详情页点击"Copy SVG"
2. 将 SVG 代码保存为 `.svg` 文件
3. 使用在线工具转换为 PNG

### SVG 转 PNG 工具

**在线转换工具：**
- https://cloudconvert.com/svg-to-png
- https://svgtopng.com/
- https://convertio.co/zh/svg-png/

**转换设置：**
- 尺寸：128px × 128px（推荐）或 81px × 81px
- 背景：透明
- 颜色：
  - 普通态：#999999（灰色）
  - 选中态：#87CEEB（蓝色）

## 批量下载技巧

### 使用 Iconify API

你可以直接通过 URL 获取不同颜色的 PNG：

```
普通态（灰色）：
https://api.iconify.design/mdi/bell-outline.svg?color=%23999999&width=128

选中态（蓝色）：
https://api.iconify.design/mdi/bell-outline.svg?color=%2387CEEB&width=128
```

**步骤：**
1. 复制上面的 URL 到浏览器
2. 右键保存 SVG 文件
3. 使用转换工具转为 PNG
4. 或者直接在 URL 中把 `.svg` 改成 `.png`（部分浏览器支持）

## 快速操作流程

1. **选择方案**：从上面4个方案中选一个
2. **访问 Iconify**：https://iconify.design/
3. **搜索图标**：输入图标名称（如 `mdi:bell-outline`）
4. **下载 SVG**：点击 Download 按钮
5. **转换 PNG**：使用在线工具转换，设置尺寸和颜色
6. **重命名文件**：按照要求命名
7. **放置文件**：复制到 `images/tabbar/` 目录
8. **编译预览**：在微信开发者工具中查看效果

---

## 注意事项

- 确保4个图标来自同一个图标库（如都用 mdi 或都用 lucide）
- 保持风格统一（都是线条风格或都是填充风格）
- 图标不要太复杂，简洁清晰最好
- 确保背景透明
