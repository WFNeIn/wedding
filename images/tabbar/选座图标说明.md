# 选座图标说明

## 图标设计

选座图标采用简洁的椅子造型，符合选座功能的直观表达。

## 图标文件

- **SVG源文件**: `seat.svg` - 矢量图标，可用于生成不同尺寸
- **需要生成的PNG文件**: 
  - `seat.png` - 用于tabBar的图标

## 生成PNG图标的方法

### 方法1: 使用在线工具
1. 访问 https://www.aconvert.com/cn/image/svg-to-png/
2. 上传 `seat.svg` 文件
3. 设置输出尺寸为 **81x81 像素**（微信小程序推荐尺寸）
4. 转换并下载 `seat.png`
5. 将文件放到 `images/tabbar/` 目录

### 方法2: 使用Figma/Sketch/Adobe XD
1. 导入 `seat.svg` 文件
2. 调整画布大小为 81x81px
3. 导出为PNG格式（@2x 和 @3x 可选）
4. 保存为 `seat.png`

### 方法3: 使用命令行工具（需要安装ImageMagick）
```bash
# 安装ImageMagick (macOS)
brew install imagemagick

# 转换SVG到PNG
convert -background none -resize 81x81 seat.svg seat.png
```

### 方法4: 使用在线图标生成器
访问 https://www.iconfont.cn/ 或 https://iconpark.oceanengine.com/
搜索"座位"或"椅子"图标，下载81x81的PNG格式

## 图标颜色

- **未选中状态**: 使用灰色 `#999999`（由app.json的tabBar.color控制）
- **选中状态**: 使用淡蓝色 `#87CEEB`（由app.json的tabBar.selectedColor控制）

图标本身应该是**单色透明背景**的PNG文件，颜色会由小程序自动应用。

## 快速方案：使用现有图标库

如果暂时没有工具生成PNG，可以：

1. 访问 https://www.iconfont.cn/
2. 搜索"座位"、"椅子"或"chair"
3. 选择一个简洁的图标
4. 下载81x81的PNG格式（单色，透明背景）
5. 重命名为 `seat.png`
6. 放到 `images/tabbar/` 目录

## 推荐的免费图标资源

- 阿里巴巴矢量图标库: https://www.iconfont.cn/
- IconPark: https://iconpark.oceanengine.com/
- Remix Icon: https://remixicon.com/
- Feather Icons: https://feathericons.com/

搜索关键词：seat, chair, 座位, 椅子

## 应用图标

生成PNG文件后，图标已经在 `app.json` 中配置好了：

```json
{
  "pagePath": "pages/seat/seat",
  "text": "选座",
  "iconPath": "images/tabbar/seat.png",
  "selectedIconPath": "images/tabbar/seat.png"
}
```

只需要将生成的 `seat.png` 文件放到 `images/tabbar/` 目录即可。
