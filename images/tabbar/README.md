# TabBar 图标目录

此目录用于存放 TabBar 导航栏图标。

## 需要的图标文件

请将以下 8 个图标文件放在此目录下:

1. `home.png` - 首页图标(未选中)
2. `home-active.png` - 首页图标(选中)
3. `details.png` - 详情图标(未选中)
4. `details-active.png` - 详情图标(选中)
5. `rsvp.png` - 回执图标(未选中)
6. `rsvp-active.png` - 回执图标(选中)
7. `blessings.png` - 祝福图标(未选中)
8. `blessings-active.png` - 祝福图标(选中)

## 图标规格

- 尺寸: 81px × 81px
- 格式: PNG
- 未选中颜色: #999999
- 选中颜色: #87CEEB

详细说明请查看: `images/TABBAR_ICONS_GUIDE.md`
