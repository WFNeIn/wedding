# 图标下载和转换详细步骤

## 一、图标尺寸选择

### 推荐尺寸：81px × 81px

**原因：**
- 微信小程序官方推荐的 tabbar 图标尺寸是 81px × 81px
- 这个尺寸在各种屏幕上显示效果最好
- 文件大小适中，不会影响加载速度

**备选尺寸：**
- 128px × 128px（如果需要更清晰）
- 64px × 64px（如果需要更小的文件）

**我的建议：使用 81px × 81px**

---

## 二、从 Iconify 下载 SVG

### 步骤1：访问 Iconify
打开浏览器，访问：https://iconify.design/

### 步骤2：搜索图标
在搜索框输入图标名称，例如：
- `mdi:bell-outline` (铃铛)
- `mdi:email-outline` (信封)
- `mdi:heart-outline` (爱心)
- `mdi:seat-outline` (座位)

### 步骤3：下载 SVG
1. 点击图标进入详情页
2. 点击右上角的 "Download SVG" 按钮
3. SVG 文件会自动下载到你的电脑

---

## 三、SVG 转 PNG（3种方法）

### 方法一：使用 CloudConvert（推荐，最简单）

**网址：** https://cloudconvert.com/svg-to-png

**步骤：**
1. 打开 CloudConvert 网站
2. 点击 "Select File" 上传 SVG 文件
3. 点击右侧的"扳手"图标进行设置：
   - Width: 81 (宽度)
   - Height: 81 (高度)
   - 勾选 "Transparent background"（透明背景）
4. 点击 "Convert" 开始转换
5. 转换完成后点击 "Download" 下载 PNG

**设置颜色：**
- CloudConvert 不能直接改颜色
- 需要在下载 SVG 前在 Iconify 设置颜色
- 或者使用方法二

### 方法二：使用 SVG to PNG Converter（可以改颜色）

**网址：** https://svgtopng.com/

**步骤：**
1. 打开网站
2. 上传 SVG 文件
3. 设置：
   - Size: 81px
   - Background: Transparent
   - Color: 可以在这里改颜色！
     - 灰色：#999999
     - 蓝色：#87CEEB
4. 点击 "Convert" 转换
5. 下载 PNG 文件

### 方法三：使用 Convertio

**网址：** https://convertio.co/zh/svg-png/

**步骤：**
1. 打开网站
2. 点击"选择文件"上传 SVG
3. 点击"转换"
4. 下载 PNG
5. 注意：这个工具不能直接设置尺寸，需要下载后再调整

## 四、更简单的方法：直接从 Iconify 获取 PNG

### 使用 Iconify API（最快捷！）

你可以直接通过 URL 获取指定颜色和尺寸的图标：

#### 格式：
```
https://api.iconify.design/{图标名称}.svg?color={颜色}&width={宽度}
```

#### 示例（方案一的8个图标）：

**首页 - 铃铛（灰色）：**
```
https://api.iconify.design/mdi/bell-outline.svg?color=%23999999&width=81
```

**首页 - 铃铛（蓝色）：**
```
https://api.iconify.design/mdi/bell-outline.svg?color=%2387CEEB&width=81
```

**回执 - 信封（灰色）：**
```
https://api.iconify.design/mdi/email-outline.svg?color=%23999999&width=81
```

**回执 - 信封（蓝色）：**
```
https://api.iconify.design/mdi/email-outline.svg?color=%2387CEEB&width=81
```

**祝福 - 爱心（灰色）：**
```
https://api.iconify.design/mdi/heart-outline.svg?color=%23999999&width=81
```

**祝福 - 爱心（蓝色）：**
```
https://api.iconify.design/mdi/heart-outline.svg?color=%2387CEEB&width=81
```

**坐席安排 - 座位（灰色）：**
```
https://api.iconify.design/mdi/seat-outline.svg?color=%23999999&width=81
```

**坐席安排 - 座位（蓝色）：**
```
https://api.iconify.design/mdi/seat-outline.svg?color=%2387CEEB&width=81
```

#### 如何使用这些 URL：

**方法A：浏览器下载**
1. 复制上面的 URL
2. 粘贴到浏览器地址栏
3. 按回车打开
4. 右键点击图标 → "图片另存为"
5. 保存为 SVG 文件
6. 再用上面的转换工具转为 PNG

**方法B：直接获取 PNG（部分浏览器支持）**
1. 把 URL 中的 `.svg` 改成 `.png`
2. 例如：`https://api.iconify.design/mdi/bell-outline.png?color=%23999999&width=81`
3. 在浏览器打开，右键保存

## 五、完整操作流程（推荐）

### 使用 CloudConvert 的完整流程：

#### 第1步：下载 SVG（4个图标）
1. 访问 https://iconify.design/
2. 搜索 `mdi:bell-outline`，下载 SVG
3. 搜索 `mdi:email-outline`，下载 SVG
4. 搜索 `mdi:heart-outline`，下载 SVG
5. 搜索 `mdi:seat-outline`，下载 SVG

#### 第2步：转换为 PNG（每个图标转2次）
访问 https://cloudconvert.com/svg-to-png

**转换铃铛图标：**
1. 上传 `bell-outline.svg`
2. 设置尺寸 81×81，透明背景
3. 转换并下载
4. 重命名为 `home.png`（灰色版）
5. 再次上传同一个 SVG
6. 转换并下载
7. 重命名为 `home-active.png`（蓝色版）

**重复上述步骤转换其他3个图标**

#### 第3步：修改颜色
由于 CloudConvert 不能直接改颜色，你需要：
- 在 Iconify 下载时就选择颜色
- 或者下载后用图片编辑工具改颜色

## 六、最简单的方法（强烈推荐）

### 使用在线 SVG 编辑器

**网址：** https://www.svgviewer.dev/

**步骤：**
1. 访问网站
2. 上传 SVG 文件
3. 在右侧面板可以：
   - 修改颜色（填充色）
   - 修改尺寸
   - 导出为 PNG
4. 设置颜色为 #999999（灰色）
5. 导出为 PNG，命名为 `home.png`
6. 再次修改颜色为 #87CEEB（蓝色）
7. 导出为 PNG，命名为 `home-active.png`

## 常见问题

### Q1: 图标颜色不对怎么办？
A: 使用 SVG 编辑器重新设置颜色后导出

### Q2: 图标太大或太小？
A: 重新转换时设置尺寸为 81px

### Q3: 图标背景不透明？
A: 确保转换时勾选了"透明背景"选项

### Q4: 找不到某个图标？
A: 在 Iconify 搜索时尝试不同的关键词，或者换一个图标库（如 lucide、ph）
