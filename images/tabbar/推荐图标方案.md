## 快速操作步骤

### 在 iconfont.cn 上操作：

1. **搜索图标**
   - 输入上面推荐的关键词
   - 选择"线条"风格（outline style）
   - 确保4个图标风格统一

2. **下载设置**
   - 点击图标进入详情页
   - 点击"下载"按钮
   - 格式：PNG
   - 尺寸：128（推荐）或 81
   - 颜色：先下载灰色 #999999

3. **批量下载技巧**
   - 可以把4个图标加入"购物车"
   - 统一下载后再分别调整颜色
   - 每个图标下载2次（灰色和蓝色）

4. **文件命名**
   ```
   铃铛图标 → home.png / home-active.png
   信封图标 → rsvp.png / rsvp-active.png
   爱心图标 → blessings.png / blessings-active.png
   座位图标 → seat.png / seat-active.png
   ```

## 备选方案：使用 Remix Icon

如果 iconfont 不方便，可以使用 Remix Icon（开源免费）：

1. 访问：https://remixicon.com/
2. 搜索并下载：
   - `ri-bell-line` → 首页
   - `ri-mail-line` → 回执
   - `ri-heart-line` → 祝福
   - `ri-armchair-line` → 选座

3. 下载 SVG 后转换为 PNG：
   - 使用在线工具：https://cloudconvert.com/svg-to-png
   - 设置尺寸 81px 或 128px
   - 设置颜色后导出

## 预览效果

下载完成后，在微信开发者工具中：
- 底部 tabbar 会显示灰色图标
- 点击某个 tab，该图标变为蓝色
- 其他 tab 保持灰色

如果效果不理想，可以随时更换图标文件。
