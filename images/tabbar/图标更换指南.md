# TabBar 图标更换指南

## 需要准备的图标

### 1. 首页图标（home）
- 关键词：home / house / 房子 / 首页
- 普通态：`home.png` - 灰色 #999999
- 选中态：`home-active.png` - 蓝色 #87CEEB

### 2. 回执图标（rsvp）
- 关键词：form / edit / 表单 / 编辑 / 回执
- 普通态：`rsvp.png` - 灰色 #999999
- 选中态：`rsvp-active.png` - 蓝色 #87CEEB

### 3. 祝福图标（blessings）
- 关键词：heart / love / message / 爱心 / 消息 / 祝福
- 普通态：`blessings.png` - 灰色 #999999
- 选中态：`blessings-active.png` - 蓝色 #87CEEB

### 4. 选座图标（seat）
- 关键词：seat / chair / 座位 / 椅子
- 普通态：`seat.png` - 灰色 #999999
- 选中态：`seat-active.png` - 蓝色 #87CEEB

## 从 iconfont.cn 下载图标步骤

### 步骤1：访问 iconfont
1. 打开浏览器访问：https://www.iconfont.cn/
2. 如果没有账号，先注册登录（可以用GitHub/微博登录）

### 步骤2：搜索并下载图标

#### 首页图标
1. 在搜索框输入"home"或"房子"
2. 找到一个简洁的房子图标
3. 点击图标，选择"下载"
4. 选择PNG格式，尺寸选择 81px 或 128px
5. 下载两次：
   - 第一次选择颜色 #999999（灰色），保存为 `home.png`
   - 第二次选择颜色 #87CEEB（蓝色），保存为 `home-active.png`

#### 回执图标
1. 搜索"form"或"表单"或"edit"
2. 找到一个表单/编辑图标
3. 下载两个版本：
   - 灰色 #999999 → `rsvp.png`
   - 蓝色 #87CEEB → `rsvp-active.png`

#### 祝福图标
1. 搜索"heart"或"爱心"或"message"
2. 找到一个爱心或消息图标
3. 下载两个版本：
   - 灰色 #999999 → `blessings.png`
   - 蓝色 #87CEEB → `blessings-active.png`

#### 选座图标
1. 搜索"seat"或"座位"或"chair"
2. 找到一个座位/椅子图标
3. 下载两个版本：
   - 灰色 #999999 → `seat.png`
   - 蓝色 #87CEEB → `seat-active.png`

### 步骤3：放置图标文件
将下载的8个图标文件放到项目的 `images/tabbar/` 目录下，替换原有文件。

## 图标规格要求

- **尺寸**：81px × 81px（推荐）或 128px × 128px
- **格式**：PNG（必须支持透明背景）
- **颜色**：
  - 普通态：#999999（灰色）
  - 选中态：#87CEEB（浅蓝色）
- **风格**：建议选择线性图标（outline style），统一风格

## 推荐图标组合

### 方案A：简约线性风格
- 首页：线条房子图标
- 回执：线条表单图标
- 祝福：线条爱心图标
- 选座：线条座位图标

### 方案B：圆润可爱风格
- 首页：圆润房子图标
- 回执：圆润编辑图标
- 祝福：实心爱心图标
- 选座：圆润椅子图标

## 其他图标资源网站

如果 iconfont 找不到合适的，可以试试：
- https://www.flaticon.com/ （需要注册，免费）
- https://iconify.design/ （开源图标库）
- https://remixicon.com/ （开源，风格统一）
- https://icons8.com/ （部分免费）

## 完成后

1. 确保8个图标文件都已放置在 `images/tabbar/` 目录
2. 在微信开发者工具中点击"编译"
3. 查看底部 tabbar 效果
4. 点击不同 tab 查看切换效果

## 注意事项

- 图标文件名必须完全匹配（区分大小写）
- 确保图标背景是透明的
- 所有图标风格要统一
- 图标不要太复杂，简洁清晰最好
