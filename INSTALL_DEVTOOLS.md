# 微信开发者工具安装指南

如果你还没有安装微信开发者工具，按照以下步骤安装。

## 什么是微信开发者工具？

微信开发者工具是开发微信小程序的官方IDE，提供了：
- 代码编辑器
- 小程序预览和调试
- 云开发管理
- 代码上传和发布

**重要**：开发微信小程序必须使用微信开发者工具，不能使用其他IDE。

## 下载安装

### 第一步：下载

访问官方下载页面：
**https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html**

根据你的操作系统选择对应版本：

#### Windows 系统
- **Windows 64位**：推荐下载稳定版（Stable Build）
- **Windows 32位**：选择对应的32位版本
- 文件大小：约 150-200MB
- 格式：`.exe` 安装包

#### macOS 系统
- **macOS**：下载 `.dmg` 文件
- 支持 Intel 和 Apple Silicon (M1/M2) 芯片
- 文件大小：约 200-250MB

#### Linux 系统
- **Linux**：下载 `.deb` 或 `.rpm` 包
- 支持 Ubuntu、Debian、Fedora 等主流发行版
- 文件大小：约 150-200MB

### 第二步：安装

#### Windows 安装步骤
1. 双击下载的 `.exe` 文件
2. 选择安装路径（建议使用默认路径）
3. 点击"安装"
4. 等待安装完成（约1-2分钟）
5. 点击"完成"

#### macOS 安装步骤
1. 双击下载的 `.dmg` 文件
2. 将"微信开发者工具"图标拖到"应用程序"文件夹
3. 打开"应用程序"文件夹
4. 双击"微信开发者工具"
5. 如果提示"无法打开"，在"系统偏好设置" → "安全性与隐私"中点击"仍要打开"

#### Linux 安装步骤

**Ubuntu/Debian**:
```bash
# 下载后安装
sudo dpkg -i wechat_devtools_*.deb

# 如果有依赖问题，运行
sudo apt-get install -f
```

**Fedora/CentOS**:
```bash
# 安装
sudo rpm -i wechat_devtools_*.rpm
```

### 第三步：首次启动

1. 打开微信开发者工具
2. 使用微信扫码登录（需要是小程序管理员或开发者）
3. 等待初始化完成

## 系统要求

### Windows
- 操作系统：Windows 7 及以上
- 内存：至少 4GB RAM（推荐 8GB）
- 硬盘空间：至少 2GB 可用空间
- 显示器：1280×800 及以上分辨率

### macOS
- 操作系统：macOS 10.11 及以上
- 内存：至少 4GB RAM（推荐 8GB）
- 硬盘空间：至少 2GB 可用空间
- 支持 Intel 和 Apple Silicon

### Linux
- 操作系统：Ubuntu 16.04+、Debian 9+、Fedora 28+
- 内存：至少 4GB RAM（推荐 8GB）
- 硬盘空间：至少 2GB 可用空间

## 导入项目

安装完成后，导入婚宴请帖项目：

### 方法一：通过界面导入

1. 打开微信开发者工具
2. 点击左侧的"+"号或"导入项目"
3. 填写项目信息：
   - **项目名称**：婚宴请帖（或自定义）
   - **目录**：选择项目所在文件夹
   - **AppID**：输入你的小程序AppID（或选择"测试号"）
4. 点击"导入"

### 方法二：直接打开

1. 在文件管理器中找到项目文件夹
2. 右键点击文件夹
3. 选择"使用微信开发者工具打开"（如果有此选项）

## 配置开发者工具

### 基础设置

1. 点击右上角的"设置"图标
2. 进入"通用设置"：
   - 编辑器字体大小：14-16px
   - 主题：可选择浅色或深色
   - 自动保存：建议开启

### 编译设置

1. 点击"详情"按钮
2. 在"本地设置"中：
   - ✅ 不校验合法域名（开发阶段）
   - ✅ 不校验 TLS 版本（开发阶段）
   - ✅ 启用调试（开发阶段）

### 云开发设置

1. 点击工具栏的"云开发"按钮
2. 如果是第一次使用，会提示开通
3. 按照提示完成云开发开通

## 常见问题

### Q1: 下载速度慢
**A**: 
- 使用国内网络下载
- 或从微信公众平台下载
- 或使用第三方下载工具

### Q2: 无法安装（Windows）
**A**: 
- 检查是否有管理员权限
- 关闭杀毒软件后重试
- 检查系统是否满足要求

### Q3: 无法打开（macOS）
**A**: 
- 在"系统偏好设置" → "安全性与隐私"中允许
- 或右键点击应用，选择"打开"

### Q4: 扫码登录失败
**A**: 
- 确保微信账号已绑定小程序
- 确保是小程序管理员或开发者
- 检查网络连接

### Q5: 导入项目失败
**A**: 
- 检查项目目录是否正确
- 检查是否有 app.json 文件
- 尝试重新下载项目代码

### Q6: 云开发无法开通
**A**: 
- 确保小程序已完成实名认证
- 确保使用的是正式AppID（测试号不支持云开发）
- 检查网络连接

## 替代方案（不推荐）

如果实在无法安装微信开发者工具，可以考虑：

### 方案一：使用在线开发工具
微信提供了网页版开发工具（功能有限）：
- 访问：https://mp.weixin.qq.com/
- 登录后在"开发" → "开发管理"中可以看到部分功能

### 方案二：委托他人开发
- 找有微信开发者工具的朋友帮忙
- 或委托专业开发者

### 方案三：使用虚拟机
- 在虚拟机中安装 Windows/macOS
- 然后安装微信开发者工具

**注意**：这些方案都不如直接安装微信开发者工具方便。
