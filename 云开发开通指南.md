# 云开发开通和数据库创建详细指南

这是一份超详细的图文指南，帮助你一步步开通云开发并创建数据库。

---

## 📱 第一步：开通云开发（5分钟）

### 1.1 找到云开发按钮

在微信开发者工具的顶部工具栏，找到并点击 **"云开发"** 按钮。

```
位置：工具栏右侧，通常在"编译"、"预览"按钮附近
图标：一朵云的图标 ☁️
```

**如果找不到"云开发"按钮**：
- 确保你已经登录微信开发者工具
- 确保项目已经正确导入
- 确保 AppID 已经配置（不能使用测试号）

---

### 1.2 开通云开发服务

点击"云开发"按钮后，会弹出一个窗口：

**第一次使用会看到**：
```
欢迎使用云开发
云开发为开发者提供完整的云端支持...
[开通] [取消]
```

**操作步骤**：
1. 点击 **"开通"** 按钮
2. 阅读并同意《微信云开发服务协议》
3. 点击 **"确定"**

---

### 1.3 创建云开发环境

开通后，需要创建一个云开发环境：

**填写信息**：
- **环境名称**：`wedding-prod`（或者你喜欢的名字，只能用英文和数字）
- **环境ID**：系统会自动生成，类似 `wedding-7g2h4m5n`
- **套餐版本**：选择 **"基础版"**（免费）

**基础版配置**（免费）：
- 数据库容量：2GB
- 存储容量：5GB
- 云函数调用次数：150万次/月
- 完全够用！

**操作步骤**：
1. 填写环境名称
2. 选择"基础版"
3. 点击 **"创建"**
4. 等待 1-2 分钟，环境创建完成

---

### 1.4 复制环境ID

创建完成后，你会看到：

```
环境创建成功！
环境名称：wedding-prod
环境ID：wedding-7g2h4m5n
```

**重要**：复制这个 **环境ID**（类似 `wedding-7g2h4m5n`），后面要用！

---

### 1.5 配置环境ID到代码

打开项目中的 `app.js` 文件，找到这一行：

```javascript
env: 'app-env-wf-invitation-3a8672c704',
```

将 `app-env-wf-invitation-3a8672c704` 替换为你刚才复制的环境ID：

```javascript
env: 'wedding-7g2h4m5n',  // 改成你的环境ID
```

保存文件。

---

## 💾 第二步：创建数据库集合（3分钟）

### 2.1 进入数据库管理

在云开发控制台（刚才打开的窗口）：

1. 左侧菜单找到 **"数据库"**
2. 点击进入数据库管理页面

你会看到一个空白页面，显示：
```
暂无集合
[添加集合]
```

---

### 2.2 创建第一个集合：rsvp（宾客回执）

**操作步骤**：

1. 点击 **"添加集合"** 按钮
2. 在弹出的对话框中：
   - **集合名称**：输入 `rsvp`（必须完全一致，小写）
   - 点击 **"确定"**

3. 创建成功后，你会看到 `rsvp` 集合出现在列表中

---

### 2.3 设置 rsvp 集合权限

**操作步骤**：

1. 点击 `rsvp` 集合名称（进入集合详情）
2. 点击顶部的 **"权限设置"** 标签
3. 在权限设置页面，选择：
   ```
   ○ 仅创建者可读写
   ○ 所有用户可读，仅创建者可写
   ● 所有用户可读写  ← 选择这个（开发阶段）
   ○ 仅管理端可读写
   ```
4. 点击 **"保存"**

**为什么选择"所有用户可读写"？**
- 开发阶段最简单，无需额外配置
- 用户可以直接提交回执
- 生产环境可以改为更严格的权限

---

### 2.4 创建第二个集合：blessings（祝福留言）

**重复上面的步骤**：

1. 返回数据库首页（点击左侧"数据库"）
2. 点击 **"添加集合"** 按钮
3. 集合名称输入：`blessings`（必须完全一致，小写）
4. 点击 **"确定"**

---

### 2.5 设置 blessings 集合权限

**操作步骤**：

1. 点击 `blessings` 集合名称
2. 点击 **"权限设置"** 标签
3. 选择：**"所有用户可读写"**
4. 点击 **"保存"**

---

## ✅ 第三步：验证配置（1分钟）

### 3.1 检查集合是否创建成功

在云开发控制台的数据库页面，你应该看到：

```
✅ rsvp        (0条记录)
✅ blessings   (0条记录)
```

---

### 3.2 测试小程序

1. 在微信开发者工具中点击 **"编译"** 按钮
2. 在模拟器中测试：
   - 打开"回执"页面，填写表单并提交
   - 打开"祝福"页面，输入留言并发送

3. 回到云开发控制台的数据库页面
4. 刷新页面，应该能看到：
   ```
   ✅ rsvp        (1条记录)  ← 有数据了！
   ✅ blessings   (1条记录)  ← 有数据了！
   ```

5. 点击集合名称，可以查看具体的数据内容

---

## ❓ 常见问题

### Q1: 找不到"云开发"按钮
**A**: 
- 确保使用的是真实的 AppID，不是测试号
- 测试号不支持云开发功能
- 需要先注册小程序账号获取 AppID

---

### Q2: 提示"环境创建失败"
**A**: 
- 检查网络连接
- 确保微信账号已实名认证
- 尝试更换环境名称（只用英文和数字）

---

### Q3: 数据提交后看不到记录
**A**: 
- 检查权限是否设置为"所有用户可读写"
- 在云开发控制台点击"刷新"按钮
- 查看开发者工具的控制台是否有错误信息

---

### Q4: 提示"cloud init error"
**A**: 
- 检查 `app.js` 中的环境ID是否正确
- 确保环境ID和云开发控制台中的一致
- 重新编译小程序

---

### Q5: 基础版够用吗？
**A**: 
完全够用！基础版配额：
- 数据库：2GB（约可存储 100万条回执记录）
- 存储：5GB（约可存储 1万张照片）
- 云函数：150万次调用/月
- 对于婚礼邀请函来说，绰绰有余！

---

### Q6: 如何查看已提交的回执和祝福？
**A**: 
1. 打开云开发控制台
2. 点击"数据库"
3. 点击 `rsvp` 或 `blessings` 集合
4. 可以看到所有记录
5. 点击"导出"可以导出为 Excel 文件

---

### Q7: 如何删除测试数据？
**A**: 
1. 进入集合详情页
2. 勾选要删除的记录
3. 点击"删除"按钮
4. 或者点击"清空集合"删除所有数据
