# 快速开始指南

这是一份面向零基础用户的完整指南，帮助你从零开始部署婚宴请帖小程序。

## 📋 准备清单

在开始之前，请确保你有：

- [ ] 一台电脑（Windows/macOS/Linux）
- [ ] 微信账号
- [ ] 3-4张婚纱照
- [ ] 婚礼的基本信息（日期、地点、联系方式等）
- [ ] 大约 2-3 小时的时间

## 🎯 完整流程（5个步骤）

### 步骤1：注册小程序账号（30分钟）

#### 1.1 访问微信公众平台
打开浏览器，访问：**https://mp.weixin.qq.com/**

#### 1.2 注册账号
1. 点击右上角"立即注册"
2. 选择"小程序"
3. 填写邮箱（用于接收验证码）
4. 设置密码
5. 验证邮箱（查收邮件，点击链接）

#### 1.3 选择主体类型
- 选择"个人"（最简单，无需营业执照）
- 或选择"企业"（需要营业执照）

#### 1.4 填写信息
- 管理员信息（姓名、身份证号）
- 手机号验证
- 微信扫码验证

#### 1.5 完成注册
- 注册成功后，记下你的 **AppID**
- 在"开发" → "开发管理" → "开发设置"中可以找到

**重要**：AppID 格式类似 `wx1234567890abcdef`，请妥善保存。

---

### 步骤2：安装开发工具（15分钟）

#### 2.1 下载微信开发者工具
访问：**https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html**

根据你的系统选择：
- Windows：下载 `.exe` 文件
- macOS：下载 `.dmg` 文件
- Linux：下载 `.deb` 或 `.rpm` 文件

#### 2.2 安装
- **Windows**：双击安装包，按提示安装
- **macOS**：拖动到应用程序文件夹
- **Linux**：使用包管理器安装

详细安装步骤请参考：[INSTALL_DEVTOOLS.md](./INSTALL_DEVTOOLS.md)

#### 2.3 登录
1. 打开微信开发者工具
2. 使用微信扫码登录
3. 确保登录的微信账号是小程序管理员

---

### 步骤3：配置项目（20分钟）

#### 3.1 下载项目代码
- 如果你是从 GitHub 下载的，解压到一个文件夹
- 记住这个文件夹的位置

#### 3.2 导入项目
1. 打开微信开发者工具
2. 点击"导入项目"或左侧的"+"号
3. 填写信息：
   - **项目名称**：婚宴请帖
   - **目录**：选择项目文件夹
   - **AppID**：输入你的 AppID
4. 点击"导入"

#### 3.3 修改 AppID
如果导入时没有填写 AppID，可以：
1. 打开 `project.config.json` 文件
2. 找到 `"appid": "your-appid"` 这一行
3. 将 `your-appid` 替换为你的实际 AppID
4. 保存文件

#### 3.4 修改婚礼信息
1. 打开 `app.js` 文件
2. 找到 `globalData.weddingInfo` 部分
3. 修改以下信息：

```javascript
weddingInfo: {
  couple: {
    groom: '新郎姓名',  // 改成你的名字
    bride: '新娘姓名'   // 改成你的名字
  },
  date: '2025-05-04',   // 改成你的婚礼日期
  time: '12:00',        // 改成你的婚礼时间
  venue: {
    name: '酒店名称',   // 改成你的酒店名称
    address: '详细地址', // 改成你的详细地址
    latitude: 30.826285,  // 改成你的纬度
    longitude: 114.811298 // 改成你的经度
  },
  contact: {
    name: '新郎',
    phone: '13900000000'  // 改成你的联系电话
  }
}
```

**如何获取经纬度？**
1. 访问：https://lbs.qq.com/getPoint/
2. 在地图上搜索你的酒店
3. 点击地图上的位置
4. 复制显示的经纬度

#### 3.5 准备婚纱照
1. 准备 3-4 张婚纱照
2. 使用 [TinyPNG](https://tinypng.com/) 压缩图片（每张 < 500KB）
3. 重命名为：
   - `20260223-183518.jpg`
   - `20260223-183532.jpg`
   - `20260223-183537.jpg`
   - `20260223-183542.jpg`
4. 放入项目的 `images/` 文件夹

---

### 步骤4：开通云开发（30分钟）

#### 4.1 开通云开发
1. 在微信开发者工具中，点击工具栏的"云开发"按钮
2. 点击"开通"
3. 同意服务协议
4. 创建环境：
   - 环境名称：`wedding-prod`（或自定义）
   - 选择"基础版"（免费）
5. 等待创建完成（1-2分钟）

#### 4.2 配置环境ID
1. 创建完成后，复制环境ID（如：`wedding-7g2h4m5n`）
2. 打开 `app.js` 文件
3. 找到这一行：
   ```javascript
   env: 'your-cloud-env-id',
   ```
4. 将 `your-cloud-env-id` 替换为你的环境ID
5. 保存文件

#### 4.3 创建数据库集合
1. 在云开发控制台，点击"数据库"
2. 点击"添加集合"
3. 集合名称输入：`rsvp`
4. 点击"确定"
5. 再次点击"添加集合"
6. 集合名称输入：`blessings`
7. 点击"确定"

#### 4.4 设置权限
1. 点击 `rsvp` 集合
2. 点击"权限设置"
3. 选择"所有用户可读写"
4. 对 `blessings` 集合重复以上步骤

---

### 步骤5：测试和发布（30分钟）

#### 5.1 本地测试
1. 在微信开发者工具中点击"编译"
2. 在模拟器中测试各个页面：
   - 首页：查看轮播图和倒计时
   - 详情页：查看婚礼流程
   - 回执页：填写表单并提交
   - 留言板：发送留言

#### 5.2 真机测试
1. 点击工具栏的"预览"按钮
2. 使用微信扫描二维码
3. 在手机上测试所有功能
4. 特别测试：
   - 地图导航是否能打开
   - 拨打电话是否正常
   - 表单提交是否成功

#### 5.3 检查数据
1. 在云开发控制台点击"数据库"
2. 查看 `rsvp` 集合，应该能看到测试提交的回执
3. 查看 `blessings` 集合，应该能看到测试留言

#### 5.4 上传代码
1. 点击工具栏的"上传"按钮
2. 填写版本号：`1.0.0`
3. 填写项目备注：`初始版本`
4. 点击"上传"

#### 5.5 提交审核
1. 登录 https://mp.weixin.qq.com/
2. 进入"版本管理"
3. 找到刚上传的版本
4. 点击"提交审核"
5. 填写审核信息
6. 提交

#### 5.6 等待审核
- 审核时间：1-7个工作日
- 审核通过后，点击"发布"即可上线

## ❓ 常见问题

### Q: 我不会编程，能完成吗？
**A**: 可以！按照本指南一步步操作即可，不需要编程知识。

### Q: 需要花钱吗？
**A**: 
- 小程序注册：免费
- 云开发基础版：免费
- 微信开发者工具：免费
- 总计：**完全免费**

### Q: 需要多长时间？
**A**: 
- 首次部署：2-3小时
- 审核等待：1-7天
- 总计：约1周

### Q: 审核不通过怎么办？
**A**: 根据审核反馈修改后重新提交，通常2-3次就能通过。

### Q: 可以修改内容吗？
**A**: 可以！修改 `app.js` 中的信息，重新上传即可。

### Q: 数据会丢失吗？
**A**: 不会！数据保存在云数据库中，永久保存。

### Q: 可以导出数据吗？
**A**: 可以！在云开发控制台可以导出数据为 Excel。
