## 💡 倒计时计算逻辑

倒计时使用 `utils/util.js` 中的 `calculateDaysDiff` 函数：

```javascript
function calculateDaysDiff(targetDate) {
  const now = new Date();
  const target = new Date(targetDate);
  
  // 将时间设置为当天的开始（00:00:00）
  now.setHours(0, 0, 0, 0);
  target.setHours(0, 0, 0, 0);
  
  const diffTime = target - now;
  const diffDays = Math.ceil(diffTime / (1000 * 60 * 60 * 24));
  
  return diffDays;
}
```

**特点**：
- 精确到天，不考虑小时和分钟
- 使用 `Math.ceil` 向上取整
- 返回正数表示未来，负数表示过去

## 🔧 技术细节

### 1. 自动刷新
- 页面加载时计算（`onLoad`）
- 页面显示时重新计算（`onShow`）
- 确保切换页面后倒计时保持最新

### 2. 状态判断
```javascript
if (daysLeft === 0) {
  countdownStatus = 'today';      // 今天
} else if (daysLeft < 0) {
  countdownStatus = 'after';      // 已过期
} else {
  countdownStatus = 'before';     // 未来
}
```

### 3. 显示优化
- 使用绝对值显示天数（避免负数）
- 根据状态显示不同文字
- 婚礼当天有特殊动画效果
