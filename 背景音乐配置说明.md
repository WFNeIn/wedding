# 背景音乐配置说明

## 功能概述

详情页添加了背景音乐功能,用户可以通过右上角的音乐按钮控制播放/暂停。

## 功能特点

### 视觉效果

1. **音乐控制按钮**
   - 位置: 右上角固定悬浮
   - 样式: 天蓝色渐变圆形按钮
   - 图标: 🎵 音符emoji

2. **动画效果**
   - 暂停状态: 静止
   - 播放状态: 旋转动画 + 脉冲波纹
   - 点击反馈: 缩放效果

### 交互功能

1. **播放控制**
   - 点击按钮: 播放/暂停切换
   - 自动循环: 播放结束后自动重新开始
   - 后台播放: 支持小程序切到后台继续播放

2. **状态管理**
   - 自动记录播放状态
   - 页面卸载时自动停止
   - 支持系统音乐控制面板

## 配置音乐文件

### 方法1: 使用网络音乐URL (推荐)

编辑 `pages/details/details.js` 中的 `playMusic` 方法:

```javascript
playMusic() {
  const manager = this.backgroundAudioManager;
  
  manager.title = '婚礼进行曲';
  manager.epname = '婚礼音乐';
  manager.singer = '婚礼背景音乐';
  
  // 替换为你的音乐URL
  manager.src = 'https://your-domain.com/wedding-music.mp3';
}
```

**音乐URL要求:**
- 必须是HTTPS协议
- 支持格式: MP3, M4A, AAC, WAV
- 建议大小: < 5MB
- 建议时长: 2-5分钟

### 方法2: 使用云存储

1. **上传音乐到云存储**
   ```bash
   # 在微信开发者工具中
   云开发 → 存储 → 上传文件
   ```

2. **获取文件URL**
   - 上传后右键点击文件
   - 选择"获取临时链接"或"获取永久链接"
   - 复制URL

3. **配置到代码**
   ```javascript
   manager.src = 'cloud://your-env-id.xxx/wedding-music.mp3';
   ```

### 方法3: 使用第三方音乐服务

可以使用以下服务托管音乐:
- 腾讯云COS
- 阿里云OSS
- 七牛云
- 又拍云

## 常见问题

### Q1: 音乐无法播放?

A: 检查以下几点:
1. 音乐URL是否正确
2. 是否使用HTTPS协议
3. 音乐文件是否可访问
4. 文件格式是否支持

### Q2: 音乐加载很慢?

A: 优化建议:
1. 压缩音乐文件大小
2. 使用CDN加速
3. 选择较短的音乐片段
4. 使用更快的服务器

### Q3: 如何禁用音乐?

A: 两种方式:
1. 用户点击按钮暂停
2. 代码中注释掉自动播放逻辑

### Q4: 音乐会自动播放吗?

A: 不会。需要用户手动点击按钮播放,符合小程序规范。

### Q5: 可以添加多首音乐吗?

A: 可以。修改代码添加音乐列表和切换功能:

```javascript
data: {
  musicList: [
    { title: '歌曲1', src: 'url1' },
    { title: '歌曲2', src: 'url2' }
  ],
  currentIndex: 0
}
```

### Q6: 如何调整音量?

A: 使用音频管理器的音量控制:

```javascript
this.backgroundAudioManager.volume = 0.5; // 0-1之间
```

## 高级配置

### 添加播放列表

```javascript
data: {
  playlist: [
    {
      title: '婚礼进行曲',
      singer: '门德尔松',
      src: 'https://example.com/music1.mp3'
    },
    {
      title: '卡农',
      singer: '帕赫贝尔',
      src: 'https://example.com/music2.mp3'
    }
  ],
  currentIndex: 0
},

playMusic() {
  const music = this.data.playlist[this.data.currentIndex];
  const manager = this.backgroundAudioManager;
  
  manager.title = music.title;
  manager.singer = music.singer;
  manager.src = music.src;
}
```

### 添加进度条

```javascript
// 监听播放进度
this.backgroundAudioManager.onTimeUpdate(() => {
  const currentTime = this.backgroundAudioManager.currentTime;
  const duration = this.backgroundAudioManager.duration;
  const progress = (currentTime / duration) * 100;
  
  this.setData({ progress });
});
```

### 添加音乐封面

```javascript
playMusic() {

  manager.src = 'https://example.com/music.mp3';
}
```

## 性能优化

### 1. 预加载音乐

```javascript
onLoad() {
  // 预加载音乐,但不播放
  const manager = wx.getBackgroundAudioManager();
  manager.src = 'your-music-url';
  manager.pause();
}
```

### 2. 缓存策略

- 小程序会自动缓存播放过的音乐
- 第二次播放会更快

### 3. 降低音质

- 使用较低比特率的MP3 (128kbps)
- 减小文件大小,加快加载
